# Set up react redux and tailwind

1. Clone the repository:

    ```bash
    git clone https://github.com/SengOudom/react_redux_depo.git
    ```

    ```bash
    cd  react_redux_depo
    ```
    

2. You can install

    ```bash
     npm install
    ```
    
    ```bash
     yarn install
    ```
 
3. You can run

    ```bash
     npm start
    ```
    
    ```bash
     yarn start
    ```
    
Open [http://localhost:3050](http://localhost:3050) to view it in the browser.
