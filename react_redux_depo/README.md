# Set up react redux and tailwind

 

1. You can install

    ```bash
     npm install
    ```
    
    ```bash
     yarn install
    ```
 
2. You can run

    ```bash
     npm start
    ```
    
    ```bash
     yarn start
    ```
    
Open [http://localhost:3050](http://localhost:3050) to view it in the browser.
